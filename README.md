## ğŸ¤– Data Process

Download dataset refer to the **Data Preparation** part in the **[GarmageNet](https://github.com/Style3D/garmagenet-impl)**.

Preprocess training data:

```bash
# Preprocess Garmageset raw objs.
python data_process/garmageset/preprocess_stylexd.py \
	--objs_dir <garmageset-root>/raw \
	--output_root <garmageset-root>/garmagejigsaw_data/preprocessed_jigsaw_train

# Prepare datalist
python data_process/garmageset/get_dataset_split.py \
	--dataset_dir <garmageset-root>/garmagejigsaw_data/preprocessed_jigsaw_train \
    --output_dir <garmageset-root>/garmagejigsaw_data/datalist
```

```yaml
  DATASET_SPLIT_DIR: "<garmageset-root>/garmagejigsaw_data/datalist"
```

Train GarmageJigsaw.

```bash
python garmage_jigsaw/train_garmage_jigsaw.py \
	--cfg experiments/train/train.yaml
```



## ğŸ­ Sewing Recovery (Inference)

Preprocess Garmage generated by GarmageNet.

```bash
python garmage_jigsaw/extract_boundary_pts.py \
    --data_dir <garmagenet-output-dir>/generated \
    --delta 0.012 \
    --shape_dedup False
```

Recovery Sewing of Garmage generated by GarmageNet.

```bash
python garmage_jigsaw/inference_ps2es.py \
    --cfg experiments/train/train.yaml \
    --weight <GarmageJigsaw-ckpt> \
    --data_dir <garmagenet-output-dir>/generated_extracted \
    --update_dis_iter 2 --inf_noise_strength 6
```



## ğŸ‘• Post-Processing

GarmageJigsaw outputs imclude a vectorized **SewingPattern**. After triangulating it and arranging it using the geometry in **Garmage**, a garment that supports physical simulation can be obtained.

### ğŸ¦¾Triangulation

Import the output of **GarmageJigsaw** into the **Style3D** software and save it as a `.sproj` file. 

You should change `data_root` to the `<garmagenet-output-dir>/garmagejigsaw_output`.

> This step **run in style3d** as a script (tool --> Python Script).

```
Copy the code of garmage_jigsaw/post_processing/json_2_sproj.py to the Style3D script window.
Change data_root in the code.
Run script in Style3D.
```

------

Export the flattened triangulated patterns from the `.sproj` file in `.obj` format. 

> This step **run in IDE**.

```bash
python garmage_jigsaw/post_processing/sproj_2_json_obj.py \
	--data_root <garmagenet-output-dir>/arrangement
```

### ğŸ¦¿Arrangement

Arrange the triangulated flattened patterns using the geometric information from generated **Garmage**.

> This step **run in IDE**.

```
python garmage_jigsaw/post_processing/arrangement.py \
	--data_dir <garmagenet-output-dir>/arrangement
```

------

Import the arranged triangulated pattern pieces together with the sewing pattern into the software.

> This step **run in style3d** as a script (tool --> Python Script).

```
Copy the code of garmage_jigsaw/post_processing/arrangement_2_sproj.py to the Style3D script window.
Change data_root in the code.
Run script in Style3D.
```
